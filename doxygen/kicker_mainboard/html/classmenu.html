Detailed Description

template<typename T>
class menu< T >

This class is for creating menus.

This class is for creating menus that can be displayed on the ssd1306 display, multiple lines are supported, and multiple data types if int or float with 2 decimal places are not what is needed, then consider using the string version of this class

Template Parameters
TThe type of the values that can be modified in the menu
