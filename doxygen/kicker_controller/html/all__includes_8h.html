enum motor_status_list

Contains the different types of states for the Odrive motor controllers.

motor_status_list has the following options: ENABLED, DISABLED, ERRORLESS, CALIBRATING, IDLE ENABLED = send messages to the motor controller to enable the motors DISABLED = motor set to idle ERRORLESS = clears motor errors CALIBRATING = motor is running it's calibration sequence IDLE = motor state does not change (if it was active it remains active, if it was disabled it remains disabled) BOOSTED = motor is in a boosted state, this is used to increase the speed of the robot when the button is pressed

◆ servo_status_list

enum servo_status_list

Contains the different types of states for the servo motors.

servo_status_list has the following options: LATCHED, RELEASED, DETACHED LATCHED = servo is in position to keep the kicking leg from moving RELEASED = servo is in position to release the kicking leg DETACHED = state servo is in on boot, will update to the expected state when controller connects
