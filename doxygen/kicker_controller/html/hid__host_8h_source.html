20/**
21 * @brief USB HID HOST string descriptor maximal length
22 *
23 * The maximum possible number of characters in an embedded string is device specific.
24 * For USB devices, the maximum string length is 126 wide characters (not including the terminating NULL character).
25 * This is a length, which is available to upper level application during getting information
26 * of HID Device with 'hid_host_get_device_info' call.
27 *
28 * To decrease memory usage 32 wide characters (64 bytes per every string) is used.
29*/
30#define HID_STR_DESC_MAX_LENGTH 32
