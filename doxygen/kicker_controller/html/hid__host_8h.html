#define HID_STR_DESC_MAX_LENGTH   32

USB HID HOST string descriptor maximal length.

The maximum possible number of characters in an embedded string is device specific. For USB devices, the maximum string length is 126 wide characters (not including the terminating NULL character). This is a length, which is available to upper level application during getting information of HID Device with 'hid_host_get_device_info' call.

To decrease memory usage 32 wide characters (64 bytes per every string) is used.
